## Config

- **Advanced** --> **Dial-up Settings** --> **Create**
  - Profile Name: IPv4
  - PDP Type: IPv4
  - APN Type: Static
  - APN: Internet
  - Authentication Type: CHAP

> Note: this is necessary to be able to reach the IPv4 WireGuard server. See [this FAQ](https://service-provider.tp-link.com/faq/780/).
