## Installation and configuration

- router mode (default)
- OpenWrt firmware
- hostname: helios
- access:
  - http://helios.lan or http://192.168.1.1
  - ssh (key auth enabled, pw auth disabled)
- after reset:
  - default mode: router
  - no pw after reset
- [WireGuard VPN server](https://openwrt.org/docs/guide-user/services/vpn/wireguard/server)
